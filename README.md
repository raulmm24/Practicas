# Practicas

Este proyecto es un sistema completo de gestión de trabajadores que he desarrollado para centralizar
y simplificar todo el flujo de trabajo dentro de una empresa: valoraciones, historial, gestión de equipos, roles y navegación limpia entre pantallas.
Mi objetivo ha sido crear una herramienta moderna, visualmente coherente y
fácil de usar, con una estética glass oscura y detalles en naranja que refuerzan la identidad del sistema.

El sistema permite gestionar trabajadores, departamentos, valoraciones y equipos desde distintos roles.
Cada usuario accede a un panel distinto según su perfil, y cada pantalla está diseñada para ser clara, rápida y funcional.

# Tecnologías utilizadas

JavaFX para la interfaz

FXML para la estructura visual

CSS personalizado para el estilo

DAO + MySQL para la gestión de datos

MVC para mantener el proyecto organizado
